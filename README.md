# Tute-Dude-DS-Assignment-3
# Module 13 - Probability Assignment

This assignment demonstrates important probability concepts using Python, NumPy, and Matplotlib through simulations and visualizations.

## Topics Covered

### 1. Basics of Probability
- Simulated coin toss experiments
- Calculated experimental probability of heads and tails
- Simulated rolling two dice
- Estimated probability of obtaining a sum of 7

### 2. Probability Estimation
- Estimated the probability of getting at least one 6 in 10 die rolls
- Used repeated trials for simulation

### 3. Conditional Probability and Bayes’ Theorem
- Simulated drawing colored balls from a bag
- Calculated conditional probabilities
- Verified Bayes’ theorem using simulation results

### 4. Discrete Random Variables
- Generated samples using a custom probability distribution
- Calculated:
  - Mean
  - Variance
  - Standard deviation

### 5. Continuous Random Variables
- Simulated an exponential distribution
- Visualized data using:
  - Histogram
  - Probability Density Function (PDF)

### 6. Central Limit Theorem
- Generated random values from a uniform distribution
- Calculated sample means
- Visualized the distribution of sample means

## Technologies Used

- Python
- NumPy
- Matplotlib
- Random Module
